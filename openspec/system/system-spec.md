# Functional Specifications - tsqlnotes

This document defines the functional scope and behaviors of `tsqlnotes`.

## Project Scope
`tsqlnotes` is a web application designed to serve as a notes management utility supporting multiple database engines.

## Core Features
1. **Multi-Database Support:** Run using SQLite, Postgres, MySQL, or MSSQL.
2. **Note Storage Operations:** Create, retrieve, update, and delete formatted note blocks.
3. **HTML5 Interface:** Dynamic template rendering using standard library features.
